# Easier_Arduino

一个整合了常见操作的Arduino库，让忍受不了坑爹的大小写敏感函数的你稍微放松点。

## 安装指南

下载zip，将Easier_Arduino文件夹放进：文档/Arduino/libraries

## 已知问题

此代码在很多的板上会无法运行，因为除Leonardo外的其他板不支持keyboard。
如遇到这种情况，请删除和Keyboard相关的函数和引用。
